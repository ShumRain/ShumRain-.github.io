个人的git配置

11 Oct 2017

  • git

由于经常使用git，我习惯把其中经常用的简化，也就是配置别名，但是每次都要去配制好的仓库里复制一份出来，而.git的文件夹又是隐藏的，所以记下来，方便实用。

s = status
c = checkout
b = branch
lg = log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold cyan)%aD%C(reset) %C(bold yellow)%d%C(reset)%n''          %C(bold red)%s%C(reset) %C(black)- %an%C(reset)' --all

这个lg是自己配的，特别喜欢，各分支的情况都能看的很清楚。
